## Overview
エクセルマクロを使ってvCardファイルの作成を行う

## Requirement

- Office 2016(Excel)
- VBA

## Usage

1. ファイルを起動する
1. 住所録に情報を書き込む
1. "メール送信"ボタンを押す
1. エクセルファイルと同じ場所に"contact.vcf"が生成される

## Features

- vCardのバージョンは3.0です

- photoは連絡先の表示画像情報です。
表示画像はJPEG形式であること、
画像の大きさは最大で幅300ピクセル、高さ300ピクセルで、ファイルサイズは50kByteを超えない事
データをbase64にエンコードする必要があります。
ファイルをbase64に変更するには下記のサービスを使ってみて下さい
https://tm-webtools.com/Tools/Base64Image
