## Overview
エクセルマクロを使ってOutlookでメール作成・送信を行う

## Requirement

- Office 2016(Outlook, Excel)
- VBA

## Usage

1. ファイルを起動する
1. "メール送信"ボタンを押す
1. Outlookが立ち上がり作成したファイルが表示される

## Features

エクセルVBAでOutlookのメールを操作するためのプログラミングに入る前に､以下の参照設定を変更しておく必要があります｡
1.VBEを開いて、「ツール」→「参照設定」
1.Microsoft Outlook 15.0 Object Libraryのライブラリにチェックを入れて、OKをクリックする
